# Default Export VS Multiple Export

## Default Export
* It is used when we are exporting single component from a file. 
* `export default componentName` - to export and,
* `import componentName from 'componentLocation' ` - to import

## Multiple Export Method
* It is used when we are exporting multiple components from a file.
* `export {component1, component2}` - to export and,
* `import {component1, component2} from 'componentLocation' ` - to import
